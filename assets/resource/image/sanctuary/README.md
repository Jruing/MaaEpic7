# 圣域图片素材说明

此目录用于存放圣域相关的图片素材。

## 目录结构

```
sanctuary/
├── sanctuary_icon.png      # 圣域入口图标
├── claim_icon.png          # 领取图标
├── claim_all_btn.png       # 一键领取按钮
├── spirit_house.png        # 精灵之家
├── spirit_claim.png        # 精灵之家领取
├── heart.png               # 心脏
├── heart_claim.png         # 心脏领取
├── tower.png               # 高塔
├── tower_claim.png         # 高塔领取
└── ...
```

## 图片要求

### 圣域入口图标 (sanctuary_icon.png)
- 主界面上的圣域入口按钮
- 尺寸建议：包含完整按钮区域
- 注意：可能与其他功能图标相邻

### 领取图标 (claim_icon.png)
- 建筑上显示的领取提示图标
- 通常为小图标或文字
- 用于定位可领取的建筑

### 各建筑图标
- 精灵之家、心脏、高塔等建筑
- 需要包含建筑的特征部分
- 避免包含动态元素

## 截图建议

1. 使用 720p 分辨率截图
2. 裁剪时保留完整的建筑/图标
3. 对于红点检测，注意颜色范围
4. 建议截取多个角度的图片

## 圣域建筑位置参考

| 建筑 | 大致位置 | 特征 |
|------|----------|------|
| 精灵之家 | 左侧区域 | 精灵图标 |
| 心脏 | 中央区域 | 心形图标 |
| 高塔 | 右侧区域 | 塔形建筑 |
| 指挥中心 | 右下区域 | 指挥图标 |

## 注意事项

- 圣域界面可能需要滑动才能看到所有建筑
- 建筑位置可能因升级而变化
- 建议定期更新图片素材
