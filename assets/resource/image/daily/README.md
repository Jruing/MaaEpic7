# 日常任务图片素材说明

此目录用于存放日常任务相关的图片素材。

## 目录结构

```
daily/
├── daily_icon.png       # 日常任务入口图标
├── claim_icon.png       # 领取图标
├── claim_all_btn.png    # 一键领取按钮
├── treasure_box.png     # 宝箱图标
├── weekly_tab.png       # 周常标签
└── ...
```

## 图片要求

### 日常任务入口图标 (daily_icon.png)
- 主界面上的日常任务入口按钮
- 尺寸建议：包含完整按钮区域
- 注意：可能与其他功能图标相邻

### 领取图标 (claim_icon.png)
- 任务列表上的领取按钮
- 通常为小图标或文字
- 用于定位可领取的任务

### 一键领取按钮 (claim_all_btn.png)
- 一键领取所有奖励的按钮
- 通常在界面顶部或底部
- 用于快速领取所有奖励

### 宝箱图标 (treasure_box.png)
- 日常宝箱图标
- 通常显示完成度进度
- 用于领取累计奖励

### 周常标签 (weekly_tab.png)
- 周常任务标签按钮
- 用于切换到周常任务界面

## 截图建议

1. 使用 720p 分辨率截图
2. 裁剪时保留完整的按钮/图标
3. 对于红点检测，注意颜色范围
4. 建议截取多个版本的图片

## 日常任务界面说明

| 区域 | 说明 | 位置 |
|------|------|------|
| 任务列表 | 显示所有日常任务 | 中央区域 |
| 宝箱进度 | 显示完成度和宝箱 | 底部或侧边 |
| 标签切换 | 日常/周常切换 | 顶部 |
| 一键领取 | 快速领取按钮 | 顶部或底部 |

## 注意事项

- 日常任务图标可能有红点状态
- 宝箱图标可能显示不同完成度
- 领取按钮可能有多种样式
- 建议定期更新图片素材
