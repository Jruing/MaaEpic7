# 讨伐副本图片素材说明

此目录用于存放讨伐副本相关的图片素材。

## 目录结构

```
hunt/
├── hunt_icon.png           # 讨伐入口图标
├── wyvern_icon.png         # 水龙图标
├── wyvern_stage_13.png     # 水龙关卡13
├── banshee_icon.png        # 木妖图标
├── banshee_stage_13.png    # 木妖关卡13
├── golem_icon.png          # 火魔像图标
├── golem_stage_13.png      # 火魔像关卡13
├── azimanus_icon.png       # 光暗图标
├── azimanus_stage_13.png   # 光暗关卡13
├── caides_icon.png         # 暗龙图标
├── caides_stage_13.png     # 暗龙关卡13
├── stage_select.png        # 关卡选择按钮
├── repeat_btn.png          # 重复挑战按钮
└── confirm_repeat.png      # 确认重复按钮
```

## 图片要求

### 讨伐入口图标 (hunt_icon.png)
- 主界面上的讨伐入口按钮
- 尺寸建议：包含完整按钮区域

### 讨伐属性图标
- 各属性讨伐的选择图标
- 需要清晰可辨认
- 避免包含动态元素

### 关卡选择图标
- 关卡 11、12、13 的选择按钮
- 建议截取关卡数字部分
- 或使用 OCR 识别

## 讨伐属性说明

| 属性 | 图标文件 | 关卡文件 | 特点 |
|------|----------|----------|------|
| 水 | wyvern_icon.png | wyvern_stage_13.png | 龙形图标，蓝色 |
| 木 | banshee_icon.png | banshee_stage_13.png | 妖精图标，绿色 |
| 火 | golem_icon.png | golem_stage_13.png | 魔像图标，红色 |
| 光 | azimanus_icon.png | azimanus_stage_13.png | 天使图标，白色 |
| 暗 | caides_icon.png | caides_stage_13.png | 恶魔图标，紫色 |

## 截图建议

1. 使用 720p 分辨率截图
2. 在讨伐界面截取各属性图标
3. 关卡选择界面截取关卡按钮
4. 战斗界面截取开始/重复按钮

## 注意事项

- 讨伐图标可能有解锁状态和未解锁状态
- 关卡选择可能需要滑动
- 重复挑战按钮可能只在战斗后出现
- 建议截取多个版本的图片以提高识别率
