# 商店图片素材说明

此目录用于存放商店相关的图片素材。

## 目录结构

```
shop/
├── shop_icon.png       # 商店入口图标
├── free_tab.png        # 免费标签
├── free_item.png       # 免费商品图标
├── buy_btn.png         # 购买按钮
└── ...
```

## 图片要求

### 商店图标 (shop_icon.png)
- 主界面上的商店入口按钮
- 尺寸建议：包含完整按钮区域
- 注意：可能与其他功能图标相邻

### 免费标签 (free_tab.png)
- 商店中的"免费"分类标签
- 或标记为免费的标签
- 用于快速定位免费商品区域

### 免费商品 (free_item.png)
- 标记为免费的商品图标
- 可以包含"免费"或"FREE"文字
- 或价格为"0"的标识

### 购买按钮 (buy_btn.png)
- 商品详情页的购买按钮
- 确认购买的操作按钮

## 截图建议

1. 使用 720p 分辨率截图
2. 裁剪时保留完整的按钮/图标
3. 避免包含动态内容
4. 对于文字标签，可以考虑使用 OCR 识别

## 注意事项

- 不同商店可能有不同的界面布局
- 免费商品可能出现在不同位置
- 建议截取多个版本的图片以提高识别率
